-- pyenv --
インストール済みバージョン
pyenv versions

インストールできるバージョンリスト
pyenv install -l

特定バージョンのpythonをインストールする(3.13.9はバージョン名)
pyenv install 3.13.9

プロジェクト内のローカルバージョンを設定
pyenv local 3.13.9


-- venv --
'venv'という名前で仮想環境を作成
python -m venv venv

仮想環境を有効化
power shell
.\venv\Scripts\Activate.ps1

コマンドプロンプト
.\venv\Scripts\activate.bat

仮想環境を終了
deactivate

-- 以下は仮想環境内で行う --
ライブラリのインストール
pip install -r requirements.txt

ライブラリインストール用のファイルを作成
pip freeze > requirements.txt
