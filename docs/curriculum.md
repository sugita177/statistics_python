

| ステップ | テーマ | 統計学の主要概念 (理論) | Pythonで使う主なライブラリと実装内容 |
| :---- | :---- | :---- | :---- |
| **1** | **データの把握と要約** | **記述統計** (平均、分散、相関、四分位数) **データの可視化** (ヒストグラム、箱ひげ図、散布図) | **pandas** (データ操作、集計) **NumPy** (基本統計量の計算) **matplotlib, seaborn** (各種グラフ作成) |
| **2** | **確率と分布** | **確率の基本** (条件付き確率、独立) **主要な確率分布** (二項、ポアソン、**正規分布**、**$t$・$\\chi^2$・$F$分布**) **中心極限定理** | **scipy.stats** (確率密度関数、累積分布関数、乱数生成) NumPy (乱数生成、シミュレーションと可視化) |
| **3** | **統計的推定** | **点推定** (不偏性、一致性) **区間推定** (母平均、母比率、母分散の**信頼区間**) | **scipy.stats** (信頼区間の計算) NumPy (推定量の計算) ※**信頼区間の意味をシミュレーションで確認** |
| **4** | **仮説検定** (最重要) | **検定の基本** (帰無仮説、P値、有意水準、過誤) **主要な検定** (**$Z$検定**、**$t$検定**、**$\\chi^2$検定** (独立性・適合度)、**$F$検定**) **ノンパラメトリック検定** (Wilcoxon, Mann-Whitney U) | **scipy.stats** (各種検定の実行、P値の取得) **statsmodels** (より複雑な検定設定) |
| **5** | **統計モデルの適用** | **線形回帰分析** (単回帰、重回帰、決定係数、残差分析) **分散分析 (ANOVA)** (一元配置、要因の有意性) | **statsmodels** (回帰モデル、分散分析モデルの構築、結果の解釈) **scikit-learn** (回帰分析の実装と予測) |
